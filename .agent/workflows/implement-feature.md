# Implement Feature Workflow

Step 1: Analyze the request and check the database schema for needed EF Core changes.

Step 2: Implement the Blazor UI frontend and the ASP.NET Core / SignalR backend.

Step 3: Trigger your build and test skills to verify compilation.

Step 4: Stage changes, write a Conventional Commit message, and push to a new feature branch.

Step 5: Use the GitHub CLI to open a Pull Request.
